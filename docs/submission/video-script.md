## หมายเหตุการถ่าย
- ใช้ PII ปลอมทั้งหมด (เช่นในไฟล์ `examples/prompts/`) — อย่าใช้ข้อมูลจริง
- ตอนเดโม ใช้ชื่อที่ทดสอบแล้วระบบจับได้ (เช่น "สมชาย ใจดี") เพื่อให้ภาพการปกปิดสมบูรณ์
- ก่อนถ่าย: เปิด backend (`run.ps1` หรือ `AIGuard.exe`) + โหลด extension ให้พร้อม
- เสียง: บรรยายไทยชัด ๆ; ใส่ซับไตเติลได้ทั้งไทย/อังกฤษ
