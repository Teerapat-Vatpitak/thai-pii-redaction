## Provider observation

Pathumma marker echo is not deterministic: a separate three-attempt observation
preserved the marker once. That is not a safety failure because a generative
answer may legitimately omit an entity. Acceptance therefore gates on a decoded
completion, no raw PII in provider-visible text, and restoration of every token
that the provider actually returns; marker preservation remains recorded
quality telemetry.
