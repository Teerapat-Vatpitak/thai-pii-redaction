## Risks & mitigations

- Signing fails in CI -> caught at draft gate; nothing public breaks; fallback engages.
- Placeholder pubkey ships by mistake -> A9 manual update-check is the tripwire.
- Booth internet dies -> rehearsed ladder (hotspot -> offline desktop loop -> video).
- Port 8000 hijacked / orphan backend -> `demo_check.ps1` before doors open.
- SmartScreen warning -> one-line note in release body + README (no paid cert).
- Low-DPI poster -> structurally eliminated (vector diagram, >=3000px shots, SVG logo),
  verified on the raster proof before printing.
